
NIEM MPD Toolkit for 3.0beta4

This package contains the following artifacts:

1.  model-package-description-3.0beta4.pdf	MPD Specification
2.  mpd-catalog-3.0.xsd				MPD catalog schema document
3.  mpd-example-conformance-assertion.docx	Example conformance assertion
4.  mpd-example-iepd-cot.zip			Example IEPD for Cursor on Target
5.  mpd-subset-3.0.zip				NIEM subset that supports MPD catalog
6.  mpd-schematron-rules-3.0.zip		MPD Schematron rule files (alpha)

Latest versions of the MPD Specification and this toolkit are always available here:

http://reference.niem.gov/niem/specification/model-package-description/
http://reference.niem.gov/niem/resource/mpd/

Ensure you obtain the version that matches the specification file.
