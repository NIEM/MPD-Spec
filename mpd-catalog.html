<annotation> ^ns
<documentation> Model Package Description (MPD) Catalog; XML document element (root) = Catalog; This schema defines a catalog.xml artifact for NIEM Model Package Descriptions (MPD). MPDs include: NIEM releases, core updates, domain updates, NIEM Information Exchange Package Documentation (IEPD), and NIEM Enterprise Information Exchange Models (EIEM). The purpose of this schema is to define/declare relatively consistent MPD content structure and metadata to process, register, search, and discover MPDs efficiently. Instances of this schema are NOT NIEM data exchanges. For this reason this schema is NOT NIEM conforming, even though type definitions and element/attribute declarations generally conform to NIEM NDR naming rules. </documentation>
</annotation>
